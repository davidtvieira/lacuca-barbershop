# Lacucaracha.pt

## Installation

Install the dependencies and start the server.

```sh
cd frontend or backend
npm ci
npm run dev
```

## Using docker-compose

Change directory to the base of the project where the docker-compose file is located.

```sh
docker-compose run
```
